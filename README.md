# 🌗 Theme Toggle App (React + useState)

A simple React.js project that demonstrates how to implement **Dark Mode / Light Mode toggle** using the `useState` hook. This project focuses on dynamic UI updates and state management in React.

---

## 🚀 Live Demo

[theme-toggle-mauve.vercel.app/] (theme-toggle-mauve.vercel.app/)

---

## 🛠 Tech Stack

- ⚛️ React.js – UI library  
- 🎣 useState – React hook for state management  
- 🎨 CSS – Styling for themes  

---

## ✨ Features

- 🌙 Dark mode toggle  
- ☀️ Light mode toggle  
- ⚡ Instant UI updates  
- 💾 Simple state-based theme switching  
- 📱 Responsive design  
- 🧩 Clean and reusable component structure  

---

## 🧠 Concept Used

This project uses the React hook `useState` to manage the current theme state.

- When user clicks the toggle button → state changes  
- UI re-renders automatically based on updated state  
- Conditional styling applied for themes
